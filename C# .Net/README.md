# C# .NET

Console application assignments built with the **.NET 10** SDK (`dotnet`).

## Assignments — Day 1

| #   | Project                                            | Description                                                  |
| --- | -------------------------------------------------- | ------------------------------------------------------------ |
| 1   | [sybageApp](Assignments/Day1/sybageApp/)           | Prints `WEL-COME TO SYBAGE`                                  |
| 2   | [averageOfMarks](Assignments/Day1/averageOfMarks/) | Accepts 5 marks and calculates their average                 |
| 3   | [sumOfMarks](Assignments/Day1/sumOfMarks/)         | Accepts 5 marks and calculates their sum                     |
| 4   | [addTwoNumbers](Assignments/Day1/addTwoNumbers/)   | Adds two hardcoded numbers                                   |
| 5   | [swapTwoNumbers](Assignments/Day1/swapTwoNumbers/) | Swaps two variables using a third variable                   |
| 6   | [asciiValue](Assignments/Day1/asciiValue/)         | Accepts a character and prints its ASCII value               |
| 7   | [netSalary](Assignments/Day1/netSalary/)           | Calculates Net Salary from Basic Salary (HRA, DA, PF, Gross) |

## Quick Start

Each folder is a self-contained console project. Run any of them with:

```bash
# from the C# .Net directory
dotnet run --project Assignments/Day1/sybageApp/sybageApp.csproj
dotnet run --project Assignments/Day1/averageOfMarks/averageOfMarks.csproj
dotnet run --project Assignments/Day1/sumOfMarks/sumOfMarks.csproj
dotnet run --project Assignments/Day1/addTwoNumbers/addTwoNumbers.csproj
dotnet run --project Assignments/Day1/swapTwoNumbers/swapTwoNumbers.csproj
dotnet run --project Assignments/Day1/asciiValue/asciiValue.csproj
dotnet run --project Assignments/Day1/netSalary/netSalary.csproj
```

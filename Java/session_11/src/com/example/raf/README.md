# Random Access File

| File | Description |
|---|---|
| [Demo.java](Demo.java) | Uses `RandomAccessFile` to seek to byte offset 14 in `input.txt` and copy all remaining bytes to `output.txt` byte by byte |
| [input.txt](input.txt) | Source file opened in read-only mode (`"r"`) |
| [output.txt](output.txt) | Destination file opened in read-write mode (`"rw"`) |
